🚀 Quiz Server Development Guide

⚠️ Important: Don't use "Launch with Live Server" on server.js! Live Server is for static files, but server.js is a Node.js Express server that needs to run differently.

🎯 Proper Ways to Start Your Server

Method 1: Using VS Code Tasks (Recommended)

Press Ctrl+Shift+P → Type "Tasks: Run Task" → Choose:

  • "Start Quiz Server" - Normal start
  • "Start with Hot Reload" - Auto-restarts on file changes
  • "Stop All Node Servers" - Stops all running servers

Method 2: Using VS Code Debug/Run

Go to Run and Debug (Ctrl+Shift+D) → Choose:

  • "Start Quiz Server" - With hot reload
  • "Debug Quiz Server" - With debugging support

Method 3: Using Terminal Commands

npm start # Start server normally npm run dev # Start with hot reload (restarts on changes) npm run stop # Stop all Node servers

Method 4: Direct Node Command

node server.js # Basic start node --watch server.js # With hot reload (Node 18+)
